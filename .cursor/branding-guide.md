# ARCLE Brand Guide

## Brand Identity

**ARCLE** — AI-powered blockchain wallet on Circle's Arc blockchain

**Brand Personality:**
- Modern & Professional
- Trustworthy & Secure
- Innovative & Tech-forward
- Accessible & User-friendly

**Brand Voice:**
- Conversational (chat-first interface)
- Clear and helpful
- Secure but not intimidating
- Professional yet friendly

---

## Primary Color Palette

### 1. Rich Blue
**Primary Brand Color**

- **HEX**: `#012AFE`
- **RGB**: `rgb(1, 42, 254)`
- **CMYK**: `(100, 83, 0, 0)`
- **Usage**: Primary CTAs, active states, key highlights, brand accents, important buttons
- **Accessibility**: Use with Onyx or white text

### 2. Onyx
**Primary Text & Dark Elements**

- **HEX**: `#111111`
- **RGB**: `rgb(17, 17, 17)`
- **CMYK**: `(0, 0, 0, 93)`
- **Usage**: Primary text, dark backgrounds, high-contrast elements, sophisticated UI components
- **Accessibility**: Excellent contrast for text on light backgrounds

### 3. Dark Grey
**Secondary Background & Neutral**

- **HEX**: `#353535`
- **RGB**: `rgb(53, 53, 53)`
- **CMYK**: `(0, 0, 0, 79)`
- **Usage**: Secondary backgrounds, subtle UI elements, borders, cards, elevated surfaces
- **Accessibility**: Good for secondary text on light backgrounds

### 4. Casper
**Light Accent & Soft Elements**

- **HEX**: `#ABBBCB`
- **RGB**: `rgb(171, 187, 203)`
- **CMYK**: `(16, 8, 0, 20)`
- **Usage**: Secondary backgrounds, borders, subtle interactive elements, disabled states, dividers
- **Accessibility**: Use for subtle accents only, not primary text

## Color Usage Guidelines

### Primary Actions
- **Buttons (Primary)**: Rich Blue (`#012AFE`) with white text
- **Buttons (Secondary)**: Transparent with Rich Blue border, Rich Blue text
- **Hover States**: Slightly darker Rich Blue
- **Active States**: Rich Blue with opacity or darker shade

### Backgrounds
- **Primary Background**: Onyx (`#111111`) - Main app background
- **Secondary Background**: Dark Grey (`#353535`) - Cards, elevated surfaces
- **Tertiary Background**: Casper (`#ABBBCB`) - Subtle accents, disabled states
- **Light Background**: White/off-white for contrast (when needed)

### Text
- **Primary Text**: Onyx (`#111111`) on light backgrounds, White on dark backgrounds
- **Secondary Text**: Dark Grey (`#353535`) or Casper (`#ABBBCB`)
- **Links**: Rich Blue (`#012AFE`)
- **Emphasis**: Rich Blue for important information

### UI Elements
- **Borders**: Dark Grey (`#353535`) or Casper (`#ABBBCB`)
- **Dividers**: Casper (`#ABBBCB`)
- **Icons**: Rich Blue for primary, Dark Grey for secondary
- **Shadows**: Onyx with opacity for depth

## Semantic Colors (Extended)

### Success (Transactions, Confirmations)
- **Primary**: Green `#10B981` (or similar)
- **Alternative**: Rich Blue for ARCLE-branded success states

### Warning (Medium Risk)
- **Primary**: Amber `#F59E0B` (or similar)
- **Usage**: Medium-risk transaction warnings

### Error/Danger (High Risk, Blocked)
- **Primary**: Red `#EF4444` (or similar)
- **Usage**: High-risk transactions, errors, blocked actions

### Information
- **Primary**: Rich Blue (`#012AFE`)
- **Usage**: Information messages, AI responses

## Typography

### Font Families
- **Headings**: Inter / SF Pro Display (bold, modern)
- **Body**: Inter / SF Pro Text (regular, readable)
- **Monospace**: For addresses, transaction hashes (Fira Code, JetBrains Mono)

### Font Sizes (Mobile)
- **H1**: 32px (Headings)
- **H2**: 24px (Section headers)
- **H3**: 20px (Subsection headers)
- **Body**: 16px (Chat messages, content)
- **Small**: 14px (Metadata, timestamps)
- **Tiny**: 12px (Labels, captions)

### Font Weights
- **Bold**: 700 (Headings, emphasis)
- **Semibold**: 600 (Subheadings)
- **Regular**: 400 (Body text)
- **Light**: 300 (Large display text)

## Spacing & Layout

### Grid System
- **Base Unit**: 4px
- **Spacing Scale**: 4, 8, 12, 16, 20, 24, 32, 40, 48, 64px

### Border Radius
- **Small**: 8px (buttons, small cards)
- **Medium**: 12px (cards, inputs)
- **Large**: 16px (modal, large cards)
- **Full**: 50% (avatars, circular elements)

### Shadows
- **Light**: `0 1px 2px rgba(17, 17, 17, 0.1)`
- **Medium**: `0 4px 6px rgba(17, 17, 17, 0.2)`
- **Heavy**: `0 8px 16px rgba(17, 17, 17, 0.3)`

## Accessibility Guidelines

### Color Contrast
- **Text on Rich Blue**: White text meets WCAG AA (4.5:1)
- **Text on Onyx**: White text for maximum contrast
- **Text on Dark Grey**: White or Casper text
- **Text on Casper**: Onyx text for readability

### Minimum Contrast Ratios
- **Normal Text**: 4.5:1
- **Large Text**: 3:1
- **Interactive Elements**: Ensure clear focus states

### Focus States
- **Focus Ring**: Rich Blue (`#012AFE`) with 2px outline
- **Alternative**: White ring on dark backgrounds

## Dark Mode (Primary Theme)

ARCLE uses a dark-first design approach:

### Base Colors
- **Background**: Onyx (`#111111`)
- **Surface**: Dark Grey (`#353535`)
- **Text**: White or Casper (`#ABBBCB`)
- **Accent**: Rich Blue (`#012AFE`)

### Dark Theme Variations
- **Background Darker**: `#0A0A0A` (if needed)
- **Background Lighter**: Dark Grey (`#353535`)
- **Surface Elevated**: Lighter Dark Grey (`#404040`)

## Brand Guidelines Summary

✅ **DO:**
- Use Rich Blue for primary actions and brand elements
- Maintain high contrast for accessibility
- Use the four core colors consistently
- Apply dark-first design approach

❌ **DON'T:**
- Use colors outside the palette without purpose
- Compromise accessibility for design
- Use Rich Blue excessively (it's a powerful accent)
- Mix too many colors in one component
